## Add your files

Clone your repo
```
git clone https://gitlab.cern.ch/<your-gitlab-username>/CompInHEP2025
cd CompInHEP2025/Ex1
create dir <yourname>
add your answers, files, subdirs (but not any data)
commit
push to your own repo
```
